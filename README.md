# ADM Kakweia — App Android

## Como gerar o APK no GitHub (passo a passo)

### 1. Cria repositório no GitHub
- Vai a github.com → "New repository"
- Nome: adm-kakweia
- Deixa privado (Private)
- Clica "Create repository"

### 2. Faz upload de TODOS os ficheiros
- Clica "uploading an existing file"
- Extrai o ZIP e arrasta TODOS os ficheiros e pastas
- Clica "Commit changes"

### 3. Activa o Actions
- Vai ao separador "Actions"
- Se aparecer botão "I understand my workflows, go ahead and enable them" → clica

### 4. Corre o workflow
- Clica em "Build APK" na lista de workflows
- Clica "Run workflow" → "Run workflow"
- Aguarda ~5 minutos (fica verde ✅)

### 5. Descarrega o APK
- Clica no workflow concluído
- Em "Artifacts" clica "ADM-Kakweia"
- Descarrega o ZIP → extrai → instala o .apk no telemóvel
